Node 开发 #

官网文档及常用库

官网文档 #

  • midwayjs - 淘宝前端架构团队，基于渐进式理念研发的 Node.js 框架
  • nestjs - 用于构建高效，可扩展的 Node.js 服务器端应用程序的框架
  • type-orm - 数据库 ORM 框架

常用库 #

  • figlet - 从文本创建 ASCII 艺术
  • commander.js - 是一个用于构建 CLI 应用程的解决方案
  • cac - 是一个用于构建 CLI 应用程序的 JavaScript 库（无依赖）
  • picocolors - 使用 ANSI 颜色进行终端输出格式化的最小和最快的库（无依赖）
  • chalk - 终端字符串样式
  • picocolors - cli-颜色文字（无依赖）
  • ora - 优雅的终端微调器（Loading）
  • cli-progress - cli-进度条
  • inquirer - 交互式命令行用户界面工具
  • fast-glob - 是一个快速批量导入、读取文件的库
  • fs-extra - 扩展了 Node fs模块，支持 Promise 和支持递归 mkdir、复制和删除。
  • js-yaml - YAML 解析器/编写器
  • qs - 支持嵌套和数组的查询字符串解析器，具有深度限制
  • shelljs - Node.js 的便携式 Unix shell 命令
  • ws - 简单易用、快速且经过全面测试的 Node.js websocket 客户端和服务器
  • cos-nodejs-sdk-v5 - 腾讯cos SDK
  • cross-env - 跨平台运行设置和使用环境变量的脚本
  • portfinder - 在当前机器上查找开放端口的简单工具
  • rimraf - 深度删除模块
  • plop - 微型生成器框架，常用于生成特定一组文件如组件开发目录
  • sucrase - 针对现代 JS 运行时时，Babel 的超快速替代品
  • esno - 是一个基于 esbuild 的 TS/ESNext 的 Node.js 运行时
  • dotenv - 可将 .env 文件 中的环境变量加载到 process.env 中
  • nodemon - 在开发 Node.js 应用程序期间使用的简单监控脚本
  • pm2 - 带有内置负载均衡器的 Node.JS 应用程序的生产流程管理器
  • semver - 语义化版本比较
  • release-it - 用于自动化版本控制和包发布相关任务的通用 CLI 工具
  • npm-run-all - 用于并行或顺序运行多个 npm 脚本的 CLI 工具
  • minimist - 是一个命令行参数解析工具
  • magic-string - 是一个用于操作字符串和生成源映射的小而快的库
  • chokidar - 是一款专门用于文件监控的库
  • debug - 是一个模仿 Node.js 核心调试技术的小型 JavaScript 调试程序
  • connect - 是一个最早期的 HTTP 服务器框架，亦可称为中间件插件
  • tsup - 是一个轻小且无需配置的，由 esbuild 支持的打包工具
  • yargs - 通过解析参数和生成优雅的用户界面，帮助您构建交互式命令行工具
