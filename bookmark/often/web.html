常用库 #

  • axios - 一个基于 promise 的 HTTP 库 (中文文档)
  • dayjs - 一个极简的JavaScript库，可以为现代浏览器解析、验证、操作和显示日期和时间
  • normalize.css - 使浏览器更一致地呈现所有元素并符合现代标准
  • nprogress - 纳米级进度条，可用于页面切换时展示进度
  • path-browserify - 为浏览器端实现了 Node.js 模块 Path
  • lodash-es - 提供模块化、性能和附加功能的现代 JavaScript 实用程序库
  • qrcode - 二维码/二维条码生成器
  • jsbarcode - 条形码生成器
  • mitt - 微型 200b 功能事件发射器
  • clipboard.js - 复制到剪贴板
  • crypto-js - 加密标准的 JavaScript 库。
  • @ctrl/tinycolor - 可根据主题色生成相关色系
  • path-to-regexp - 将路径字符串等/user/:name转换为正则表达式
  • validator.js - 字符串验证器，常应用于表单验证
  • js-base64 - Base64转码器

网页端 #

  • element-resize-detector - 优化了元素的跨浏览器调整大小侦听器。比相关方法快 37 倍

  • sortablejs - 是一个用于可重新排序的拖放列表的 JavaScript 库。
  • vue.draggable.next - 基于 SortableJS 封装的 Vue 组件，允许拖放并与视图模型数组同步
  • jqx-grid - Grid 是一个功能强大的 jQuery 小部件，解决大数据报表数据展示表格方案
  • v-contextmenu - Vue.js 的右键菜单组件
  • wangeditor - 开源 Web 富文本编辑器，开箱即用，配置简单
  • jspreadsheet - 一个轻量级的 javascript 插件，用于创建在线交互式电子表格
  • web-data-rocks - 是一个交互式数据透视表，用于数据分析和可视化
  • vue-json-editor - Vue.js 的 json 编辑器
  • vue-waterfall - Vue.js 的瀑布布局组件
  • vue-cropperjs - Vue.js 的裁剪图片组件
  • print-js - 一个小型 javascript 库，可帮助从 Web 打印
  • xlsx - 可以读取电子表格文件并以各种格式导出内容
  • screenfull - 用于跨浏览器使用全屏模式，可将页面或任何元素变为全屏
  • monaco-editor - VS Code的代码编辑器
  • sheet.js - 简化的电子表格
